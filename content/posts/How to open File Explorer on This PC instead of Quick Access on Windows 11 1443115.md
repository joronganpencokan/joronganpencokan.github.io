Introduction:

If you're someone who has just upgraded to Windows 11 and are struggling to open File Explorer, you're not alone. The process of opening File Explorer in Windows 11 is slightly different than previous versions of Windows, leading many users to believe that they're doing something wrong. In this article, we'll reveal the secret trick to opening File Explorer on Windows 11 so that you can save your time and make your experience more comfortable.

Step-by-Step Guide:

Here's a step-by-step guide to opening File Explorer:

Step 1: Click on the Windows button or press the Windows key on your keyboard.

Step 2: On the menu that appears, click on the "Documents" option.

Step 3: In the Documents window, click on the "This PC" option located on the left pane.

Step 4: You'll now see all drives and folders on your computer. Here, you can select the folder that you want to access and double-click to open it.

Alternate Method:

If you'd like a quicker method, here's an alternate way to open File Explorer:

Step 1: Click on the Windows button or press the Windows key on your keyboard.

Step 2: Type "File Explorer" in the search bar and click on the corresponding search result.

Step 3: The File Explorer window will now open, and you'll see all drives and folders on your computer.

Conclusion:

On Windows 11, it’s possible to open the “This PC” page instead of the “Quick Access” page by changing the Folder Options settings. By default, File Explorer opens “Quick Access,” which contains your pinned folder locations and recent files. However, you can change the default settings to open the “This PC” page for quicker access to your profile folders, drives, and network locations.
 
In this guide, you will learn the steps to start on This PC by default when opening File Explorer on Windows 11.
 
## Open File Explorer on This PC instead of Quick Access
 
To change the default start page to This PC in File Explorer, use these steps:
 
- Open File Explorer on Windows 11.
 - Click the See more (three-dotted) drop-down menu and select Options.
 - Click the General tab.
 - Use the “Open File Explorer to” drop-down menu and select the This PC option.
 - Click the Apply button.
 - Click the OK button.

 
Once you complete the steps, the next time you open File Explorer, it will open on “This PC” instead of the Quick Access page.
