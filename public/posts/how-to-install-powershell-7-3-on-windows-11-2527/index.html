If you’re a Windows 11 user, you already know the importance of Powershell. This powerful scripting language can automate your tasks and save you valuable time. With the release of Powershell 7.3, it’s even more important to keep up with the latest version. In this article, we’ll show you just how easy it is to install Powershell 7.3 on your Windows 11 computer.

Why Install Powershell 7.3?#

Before we dive into the installation process, let’s take a quick look at why you should upgrade to Powershell 7.3. This version comes with a number of improvements, including:

  • Improved performance and reduced memory usage
  • Better compatibility with Azure and other cloud services
  • New operators and cmdlets for improved scripting
  • Enhancements to debugging and error handling

Overall, Powershell 7.3 is faster, more efficient, and more powerful than previous versions. Plus, it’s fully supported on Windows 11, so you can rest assured that it will work seamlessly on your computer.

How to Install Powershell 7.3#

Now that you know the benefits of Powershell 7.3, let’s get to the installation process. Fortunately, it’s incredibly easy to install Powershell 7.3 on Windows 11.

  1. Open your browser and navigate to the official Powershell 7.3 download page (https://github.com/PowerShell/PowerShell/releases/tag/v7.3.0).

  2. Scroll down until you see the “Assets” section.

  3. Click on the “download here” link next to the Powershell 7.3 package for your version of Windows 11.

  4. Once the download is complete, open the downloaded file.

  5. Follow the on-screen prompts to complete the installation.

That’s it! You now have Powershell 7.3 installed on your Windows 11 computer.

Conclusion#

Powershell is an essential tool for Windows 11 users, and upgrading to Powershell 7.3 should be a top priority. Thankfully, the installation process is incredibly simple and straightforward. Just follow the steps outlined above, and you’ll have Powershell 7.3 up and running in no time. With this powerful scripting language at your fingertips, your productivity will skyrocket, and you’ll be able to automate your tasks like never before.

On Windows 11, PowerShell 7.3 is the next version of the command-line shell and scripting language designed for advanced users to automate tasks and change system settings.

PowerShell 7.2 is still the current LTS (3-year supported) release cycle. This release also includes improvements for tab completion, error handling, session and remoting, and updated cmdlets, among other changes.

Since it’s a cross-platform application, you can download and install the new version of PowerShell on Windows 11, 10, 8.1, Windows Server (2012 R2 and later), Ubuntu, Debian, CentOS, Fedora, and Alpine for Linux, and macOS. Install PowerShell 7.3 on Windows 11#

To install PowerShell 7.3 on Windows 11, use these steps:

  • Open PowerShell 7.3 on GitHub.
  • Under the “Assets” section, click the PowerShell-7.3.0-win-x64.msi.
  • Save the file on the device.
  • Double-click the downloaded file to launch the installer.
  • Click the Next button.
  • (Optional) Select the installation folder.
  • Click the Next button.
  • (Optional) Check the “Add ‘Open here’ context menu to Explorer” option.
  • (Optional) Check the “Add ‘Run with PowerShell 7 context menu’ for PowerShell files” option.
  • Click the Next button.
  • Check the “Enable updating PowerShell through Microsoft Update or WSUS” option.
  • Check the “Use Microsoft Update when I check for updates” option.
  • Click the Next button.
  • Click the Install button.
  • Click the Finish button.

Once you complete the steps, the new version of PowerShell will install on the computer. After the installation, it’ll be available from the Start menu and desktop and File Explorer right-click context menu.

Install PowerShell 7.3 from Microsoft Store#

To install PowerShell 7.3 through the Microsoft Store, use these steps:

  • Open Microsoft Store on the web.
  • Click the Get in Store app button for PowerShell 7.3.
  • Click the Open button.
  • Click the Install button from the Microsoft Store.

After you complete the steps, you can click the Launch button or open the app from the Start menu to start using PowerShell.

Install PowerShell 7.3 from winget command#

To install PowerShell 7.3 quickly with the winget command, use these steps:

  • Open Start.
  • Search for Command Prompt, right-click the top result, and select the Run as administrator option.
  • Type the following command to install PowerShell 7.3 on Windows 11 with winget and press Enter:
  • winget install –id Microsoft.PowerShell

Once you complete the steps, the Windows Package Manager will download and install the latest version of PowerShell on your computer.
